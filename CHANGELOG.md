# Changelog

All notable changes to the "MiniNFT" project will be documented in this file.

## [Unreleased]

### Added
- **Wooden Theme**: Complete UI overhaul with organic colors, textures, and animations.
  - `woodGrain`, `treeSway`, `warmGlow`, `emberFloat`, `smokeRise` animations.
  - Custom wooden scrollbars and selection colors.
  - Carved wood button styles.
- **Scripts**:
  - `generateWallets.js`: Generate 250 EVM wallets.
  - `interactionScript.js`: Batch minting and balance checking.
  - `verify_setup.js`: Environment verification.
- **Contract**:
  - Increased supply to 1005.
  - Reduced price to 0.000001 ETH.

### Changed
- Updated `MAX_SUPPLY` from 505 to 1005.
- Updated `MINT_PRICE` from 0.00001 ETH to 0.000001 ETH.
- Replaced all emojis with nature/wood themed icons.
